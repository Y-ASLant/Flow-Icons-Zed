基于 [Flow Icons](https://open-vsx.org/extension/thang-nm/flow-icons) 插件移植到 Zed 编辑器的图标主题。

## 使用方法

1. 克隆本仓库：

```bash
git clone https://github.com/Y-ASLant/Flow-Icons-Zed.git
```

2. 在 Zed 编辑器中，打开命令面板（`Ctrl + Shift + P` 或 `Cmd + Shift + P`）
3. 搜索并选择 **Install Dev Extension**
4. 选择克隆下来的 `Flow-Icons-Zed` 文件夹
5. 安装完成后，进入设置（`Ctrl + Shift + P` ），在 `icon theme` 中选择想要的主题：

```json
{
  "icon_theme": {
    "dark": "Flow Dawn",
    "light": "Flow Dawn Light"
  }
}
```

### 可用主题

| 主题名称 | 暗色 | 亮色 |
|---------|------|------|
| Dawn | `Flow Dawn` | `Flow Dawn Light` |
| Deep | `Flow Deep` | `Flow Deep Light` |
| Dim | `Flow Dim` | `Flow Dim Light` |
| You | `Flow You` | `Flow You Light` |
